# w3c-standards
https://www.w3.org/standards/

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/cnpedia/w3c-standards/issues)！

## How to use?

开发的时候运行下列命令：

```
npm run dev
```

开发完毕需要发布的时候，请运行下列命令：

```
npm run build
```

当然，你也可以[在线访问](https://cnpedia.github.io/w3c-standards)哦~
